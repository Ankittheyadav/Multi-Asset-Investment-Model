# Multi-Asset-Investment-Model
A Flask-based web application designed to educate users on the benefits of diversification in investments.
## Overview  
The Multi-Asset Investment Model is a Flask-based web application designed to educate users on the importance of investment diversification in India. The application provides detailed insights into six major asset classes — **Equity**, **Mutual Funds**, **Real Estate**, **Gold**, **Fixed Deposits**, and **Public Provident Fund (PPF)** — guiding users who have little to no knowledge about investments. The project aims to help users make informed decisions by understanding the potential returns and risks associated with different investment options.  

## Key Features  
- **Diversification Guidance**: Educates users about the benefits of investing across multiple asset classes.  
- **Asset Class Pages**: Dedicated sections for Equity, Mutual Funds, Real Estate, Gold, Fixed Deposits, and PPF, each providing relevant details.  
- **Growth Calculators**:  
  - Equity: Projects future returns based on a 12% average annual growth rate.  
  - Mutual Funds: Calculates growth with a 10% average annual return.  
  - Real Estate: Offers options for Residential, Commercial, and Rental Yields with varying growth rates.  
  - Gold: Estimates growth using an 11% average return.  
  - Fixed Deposits: Adjusts returns based on investment tenure (<1 year: 6%, 1–3 years: 7%, >3 years: 8%).  
- **User-Friendly Interface**: Simple and responsive web design for seamless navigation and usage.  
- **Real-Time Calculations**: Inputs investment data and duration to calculate projected returns and profits.  

## Tools and Technologies  
- **Flask**: Backend framework for building the web application.  
- **HTML & CSS**: For responsive and user-friendly frontend design.  
- **JavaScript (JSON)**: Handles data exchange between the frontend and backend.
